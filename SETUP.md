# ProfitReserve Setup Instructions

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/marriagadakeim/ProfitReserve.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd ProfitReserve
   ```

3. **Install dependencies:**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

## Setup

1. **Environment Variables:**
   Create a `.env` file in the root directory of the project and add the following configurations:
   
   ```plaintext
   PORT=3000
   DB_URL=your_database_url
   JWT_SECRET=your_jwt_secret
   ```

2. **Start the Application:**
   You can start the application using:
   ```bash
   npm start
   ```

3. **Accessing ProfitReserve:**
   Open your web browser and go to `http://localhost:3000` to access the dashboard.

## Troubleshooting

- Ensure all dependencies are correctly installed.
- Check the environment variables to make sure they are set up properly.
